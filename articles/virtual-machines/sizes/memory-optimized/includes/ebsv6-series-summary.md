# Customer intent: As a cloud architect, I want to assess the Esv6-series virtual machines' specifications, so that I can determine their suitability for memory-intensive enterprise applications and optimize our infrastructure for performance.
---
The Ebsv6-series virtual machines run on the 5th Generation Intel® Xeon® Platinum 8573C (Emerald Rapids) processor and deliver higher remote storage performance compared to the previous generation. The Ebsv6-series is ideal for storage throughput intensive workloads. These virtual machines offer up to 800,000 IOPS and 14000 MBps of remote disk storage performance. Additionally, the Ebsv6 VM series offer up to 192vCPU and 1832 GiB of RAM providing larger processing power to consolidate workloads. Refer to the Premium SSD, Premium SSD v2, and Ultra disks to attain published disk performance.
